## 操作方式

### 命令行
  参考文档
### windows便捷输入框
1. 双击启动higo-dog.exe文件
2. 在窗口模式切换到英文输入法，然后最小化到后台运行即可使用
3. 双击Ctrl可呼出输入框，输入命令（省略higo前缀），例如输入show chrome
